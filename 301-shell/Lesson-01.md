### I. `if`

`gedit 01-if-file`

`./01-if-file`

*Note it says "Yes." because "nothing" technically exists*

`./01-if-file myfile`

*Note it says nothing because "myfile" does not exist*

`ls`

`touch myfile`

`ls`

`./01-if-file myfile`

*Note the response because the file exists*

`gedit 01-if-dir`

`./01-if-dir mydir`

`mkdir mydir`

`ls`

`./01-if-dir mydir`

`./01-if-file otherfile`

`./01-if-dir otherdir`
